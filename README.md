# Trainwise AI

Statisk hemsida för AI Agency Portfolio-uppgiften. Sidan innehåller:

- träningsguide som rekommenderar schema
- träningsscheman för olika mål, nivåer och utrustning
- övningsbilder, video och ljud
- fiktivt kundcase
- rapportdel om hur AI användes
- länkar till uppgiftsmaterial och bedömningsmatris

## Öppna lokalt

Öppna `index.html` direkt i webbläsaren, eller kör en lokal server:

```powershell
node local-server.cjs
```

Besök sedan `http://localhost:8088`.

## Publicera på GitHub Pages

1. Skapa ett nytt repository på GitHub.
2. Lägg upp alla filer i den här mappen: `index.html`, `styles.css`, `script.js`, `assets/`, `docs/` och PDF-filen i roten.
3. Gå till repositoryts `Settings`.
4. Välj `Pages`.
5. Under `Build and deployment`, välj `Deploy from a branch`.
6. Välj branchen `main` och mappen `/root`.
7. Spara. GitHub visar en länk när sidan är publicerad.

Kontrollera att länken går att öppna utan inloggning innan du lämnar in den.
